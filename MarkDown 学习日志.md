### 有序列表

 1. 打开冰箱
 2. 把大象放进去
 3. 关上冰箱

### 无序列表
- 短线 -
- 尽量不要重复

* 这样也可以 
* 短线和✳号

### 任务列表
- [ ] 是这样吗
- [ ] 每一个符号都要用空格隔开
- [ ] 只有两种状态 不能加字
- [x] 之后必须跟随语句- [空格]和 - [x] 两种形式
- [x] 上一个是勾选下一个也是勾选
